# Dashboard de Vendas de Carros Usados

Este projeto utiliza **Streamlit**, **Pandas** e **Plotly Express** para criar um dashboard interativo que permite explorar dados de anúncios de veículos usados.

## Funcionalidades
- Visualizar um **histograma** da quilometragem (`odometer`)
- Visualizar um **gráfico de dispersão** entre preço e quilometragem
- Interface interativa feita com **Streamlit**

## Como executar
1. Instale as dependências:
   ```bash
   pip install streamlit pandas plotly
